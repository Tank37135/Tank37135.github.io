选择你的电脑系统

首先，我们了解一下世界主流的电脑系统

  • Windows(微软)
    Windows 是全球使用最广泛的桌面操作系统。它由微软公司开发，常用于个人电脑、企业工作站和笔记本电脑。Windows 系统易于使用，支持大量的软件和硬件，兼容性高，也就是说，你对这台电脑的配置不满，你可以通过更换几乎任意厂家的芯片来升级此电脑。
    因为其使用最广泛，所以拥有最大的软件库，几乎所有软件都支持Windows，假如系统出现问题，可以很轻松地在网络上找到解决方案，微软官网售卖的电脑有超高的颜值，画在屏幕背面的中央的商标显得电脑非常气派。
    如果你希望获得好用，但这个软件比较小众(很多大众软件也不支持其他系统)，Windows是最好的选择
  • macOS (苹果)
    macOS 是苹果公司开发的操作系统，专门用于其Mac系列电脑。macOS 以其简洁、稳定而著称。它也提供了优异的图形和多媒体处理能力，常用于创意行业如设计、视频编辑等。
    但macOS系统封闭，对硬件兼容性不高，只能通过更换特定厂家的芯片升级配置，软件市场小，尤其是游戏，能支持的游戏(2024统计)可以说两只手就可以数过来
    和Windows一样，苹果官方的电脑颜值高，但macOS的动画优化优于Windows(视觉体验更好)，不过系统故障很难找到解决办法，目前(2024)据我了解，主要只有苹果自己售卖macOS
    但由于macOS软件市场远小于Windows，所以可以使用macOS系统来避免很多风险软件的安装(对于见过Windows的用户会很不习惯，比如其系统关闭浏览器要点击左侧红色点然后再任务栏右键图标，点击退出而不是点击右侧叉号，注意，macOS系统的鼠标滚轮是反的)
    对了，买这种系统之前记得先买一个转换器和一个扩展坞，因为不进行复杂的办公，所以一般不需要买鼠标。
  • Linux
    Linux 是一个开源的操作系统，广泛应用于服务器、嵌入式设备以及开发者和技术爱好者的个人电脑上。Linux 有许多不同的发行版（如 Ubuntu、Debian、Fedora、CentOS 等），并因其高定制性和强大的安全性在开发和服务器领域特别受欢迎。
    对于专业人士非常适合，但软件市场小(支持的软件比macOS多点，但仍然远远小于Windows)，系统没有官方售卖的电脑，最好在Windows系统上下载此系统，然后安装
  • BSD 系统（如 FreeBSD、OpenBSD、NetBSD）
    BSD 是一种类 Unix 操作系统，基于 Berkeley Software Distribution（BSD）。它具有高度的稳定性和安全性，广泛应用于服务器、路由器、嵌入式系统等领域。

以上是常见的四类电脑系统，如果你不知道改选什么，就直接选择Windows
如果你是仅看电视，使用基本的办公，且对视觉要求高，就选择macOS(屏幕好)。如果你专业开发或其他专业人士，选择Linux(推荐)或者BSD
如果都不是，选择Windows，否则你将放弃很多便利

当然，也可以组合使用，推荐使用Windows台式机(处理复制问题，当然，看电视，画图，剪辑等简单问题也可以)，加macOS笔记本(看电视，画图)。如果愿意，可以再准备一台电脑(笔记本台式都可以)使用Linux

鼠标

鼠标硬件篇

我们重点了解三个核心元件

首先是微动，微动是鼠标左右键下的元件，用来检测用户是否点击，假如你的鼠标按键失灵，可以选择更换微动，不一定要换整个鼠标

接着是滚轮，滚轮主要分两种，一种是"光栅式"，结构复制，价格较贵，但耐用，稳定，仅微软，罗技等少数企业生产。
另一种是"机械编码式"，其结构简单，便宜，但不耐用(一般是你的微动先坏，放心)。

传感器：传感器主要是用来识别所在位置，然后转换成数据汇报给计算机转换成光标移动的
对传感器的影响主要是鼠标垫，一般是编制布垫。当然也有好的传感器，比如微软蓝影技术(新一代鼠标追踪技术)，几乎可以用于任何地方。
鼠标传感器有两个参数。一个叫做采样率(CPI)。有人将其称为DPI(图像每英寸长度内的像素点数。)，但其实应该叫做CPI(英寸采样率)。通俗的讲，CPI越高，鼠标的灵敏度就越高。在现实中鼠标在鼠标垫上移动单位距离，CPI越大的鼠标在屏幕上移动的距离越大。CPI不是不能调，在系统的控制面板>硬件>鼠标选项 可以修改。 第二个参数叫做回报率，意思是指鼠标向计算机报告位置的频率。一般情况下频率越高，相对于的延迟越小。但过高的回报率与较高的回报率基本没有感受上的区别。

有些鼠标就有鼠标侧键。这些鼠标侧键跟键盘差不多,不同的鼠标侧键有不同的功能。但在游戏时使用这些鼠标侧键很容易被判定为作弊。

鼠标软件知识

鼠标左键在电脑中一般可以作为选择，比如你使用鼠标左键点击某一文件的时候，可以选中该文件。
鼠标右键可以理解为属性，在很多时候对着某东西使用鼠标右键会出现右键菜单。这些菜单里面一般提供了很多关于该文件属性的选项。
鼠标中键及鼠标滚轮，一般用作对过长的文件进行滚动。

鼠标左键选中文本之后拖动文本到某一可输入文本的地方，系统会复制该文本。(对于一些不允许复制的网站有效)。用鼠标将网站上的图片拖至文件夹或桌面(桌面也是一种文件夹。)，系统会下载该图片。在浏览器中，拖动网址进入输入框会直接填入该网址，拖入上方标签也会替换标签页或在标签页旁边插入新的标签页。

在浏览器上对着链接使用鼠标中键，可以使链接在后台新建标签页中打开。
在有标签页的软件上对着标签页使用鼠标中间可以快速关闭标签页。
使用Alt+Tab多任务视图时，可以使用鼠标中键对着任务视图单击关闭窗口。
在允许滚动的位置单击鼠标中键可以实现自动滚动，滚动的方向由你鼠标的移动方向决定。

鼠标组合键

该内容需要用到一点点键盘知识，你只需要知道这些按键在哪里就行。
左键

对文件使用双击，可以打开该文件。对文字使用双击，可以选中该词语。对文字使用三击，可以选中整行的文字。

在大多数情况下，你可以按住鼠标左键不放，然后拖动鼠标，就可以选中一片区域的文字或文件。也可以先对着起点单击鼠标左键。然后按住shift不放，最后在末尾单击鼠标左键，也可以达到选择一片区域的效果。

按住ctrl再使用鼠标单击链接，可以使链接在后台新建标签页中打开。

按住Alt在使用鼠标可以选中网站链接上的文字。

中键

按住ctrl使用鼠标滚轮滚动，可以在支持放大缩小的地方放大和缩小页面。

右键

按住shift加鼠标右键可以显示全部菜单。(contextmenumanager可以帮助你管理右键菜单。)

鼠标手势

鼠标手势是指使用鼠标按照特定的键位在滑动特定的轨迹实现特定的操作。

系统本不支持鼠标手势。但有些软件支持，比如说很多浏览器，部分输入法。也有专业的鼠标手势软件(mouselnc等)。
Quicker是一个强大的鼠标工具软件。提供了很多便捷功能。

键盘

其实，在计算机中键盘是最重要的控制硬件，作用远远大于鼠标。我们可以通过键盘将键盘中预先设置好的字符，指令，数据传输到电脑系统中。从而向电脑发出命令、传入数据等

硬件知识

PCB电路板（主控芯片+电路）是键盘的核心，主控芯片识别按下的键，可以通过主控芯片对键盘灯光控制。
电路是识别操作的基础，部分键盘可以通过PCBA的灯座增加灯光，或者出厂预制RGB背光（可通过主控调节模式，可以通过驱动自定义）

以前键盘常采用键线不分离，但现在也有使用type—C接口（键线分离）

键轴

键轴是表现键盘手感最直观的感受。
根据键盘工作方式的不同分类：

  • 机械键盘

    有独特而多样的手感，寿命长大部分可以单键更换

  • 薄膜键盘

    价格低，防水性好，噪音低，使用时间长后键盘松垮，按键可能无法复原
    不完全编码式键盘，矩阵识别，多按键共一条信号线，这几个按键同时按键会起冲突，只会输出一个按键

  • 静电容键盘

    小众，价格昂贵，完全静音，损耗极低

键帽

键帽也影响手感（通过材质影响）

  • ABS苯乙烯树脂，容易打油，便宜
  • PBT聚对苯二甲酸丁二醇脂，耐油，但贵

主要根据自己喜欢选择不同键盘，因为我们主要讲电脑使用，使用想要更多知识可以去搜搜

使用

键盘也分40%-80%键盘

注册微软账户需要邮箱，假如你还没有邮箱，添加你直接点获取新的电子邮件地址，然后微软会帮你带到outlook邮箱的注册界面。微软outlook简洁，高效，速度快，安全性好。是一款很不错的邮箱。
注册完邮箱之后需要绑定电话号码或者是副邮箱，电话号码或者是副邮箱用来防止你的邮箱丢失。你可以暂且选择跳过，但在七天之内必须绑定，否则你的邮箱被判为违规并且封禁。
当他让你选择你所在的国家和地区的时候，如果你希望获得一些国外并且不支持国内的东西，你可以选择国外。最后在经过反AI系统就可以拥有微软账户。(如果你通过不了反AI系统，你可以尝试刷新网站以更换题目。)

注册完微软账户之后就要遵守微软账户的守则。这些数字在你注册账号的时候有写到。微软账号至少要每三年登陆一次。(这个不用担心，你每天登录电脑的时候就是登录一遍账户。)至少每一年登陆一次outlook或者是onedrive，除非你没有。否则账户违规。

你在电脑上登录你的账户

在你刚买电脑准备登录账户的时候，它会有提示，你只要跟着提示一步步操作即可登录，你假如你已经买完电脑，并且你的电脑上没有登录你的微软账户，你可以使用win+I打开设置，假如你是系统内置管理员账户，在设置的上方会提示你登录，假如你不是内置管理员，你可以点击账户，然后登录。

注销账户

当你不想要你的微软账户的时候，我建议你给自己三天时间冷静一下。因为系统不会给你时间冷静，如果你执意选择注销，你可以点击你在任意已经登录的微软的界面的头像，然后下拉菜单就会有一个注销账户。点击它之后你的微软然后会被立刻注销。没有冷静的时间。(你别问我是怎么知道的，呜呜呜)

本地账户

权限等级

  1. 管理员账户

    管理员账户是你计算机内权限等级最高的账户。在一般情况下，新的电脑会自动帮你创建一个管理员账户。
    管理员账号具有系统的全部控制权。可以安装或卸载软件、修改系统设置、创建或删除用户账户、访问所有文件和文件夹。能够管理其他用户账户并赋予其不同的权限。可以执行需要高级权限的操作，通常系统提示管理员确认后才会执行相关操作。注意:电脑上至少要保证有一个管理员在正常工作。

  2. 标准账户

    你标准账号的权限等级相对于管理员比较低。不能安装或卸载程序，不能修改系统设置，也不能管理其他用户账户。对系统文件和文件夹的访问有限，主要可以访问自己创建的文件。在执行需要管理员权限的操作时，系统会要求输入管理员账户的凭证（如密码）。
    在日常生活中，你可以使用标准账户而不是管理员账户。这样可以更有效的保证电脑安全，但在某些情况下，也会降低工作的效率。

  3. 来宾用户

    这种用户的权限等级最低，一般是远程联机时的用户一般情况下，系统默认关闭此账户，或者被限制成非常基础的操作。你可以通过控制面板>账户 或者控制面板>远程操控 来开启来宾账户
    默认情况下，来宾账户不能安装或卸载软件，也不能修改系统设置。只能访问部分共享文件或公共文件夹，无法访问其他用户的个人文件。

创建本地账户

创建本地账户有很多种方式，本篇主要介绍通用的方式(其他方式游戏的非专业版系统使用不了。)。

注意，只有管理员账户才能创建账户。首先 设置>账户>家庭和其他账户 然后选择新增账户，先选择账户的类型，然后再选择是否要设置密码，如果不输入密码，默认为不设密码。创建账户的过程中，你可以选择一个创建一个没有微软账户的账户(也就是一个本地账户，这种账户可以不需要微软账号登录，同样也只有你的电脑上可以登录。)。你当你是管理员的时候，你可以重新选择该账户的类型。

预备知识

本页介绍一些基础概念和简单知识点。该页面不定时更新(具体是否更新可以看前言)

内存单位

  1. 比特（bit，简写为b）
    最小的存储单位，表示数据的二进制状态（0或1）。一般情况下用不到。
  2. 字节（Byte，简写为B）
    1字节 = 8比特（b）。
  3. 千字节（Kilobyte，简写为KB）
    1KB = 1024B
  4. 兆字节（Megabyte，简写为MB）
    1MB = 1024KB = 1,048,576B。
  5. 吉字节（Gigabyte，简写为GB）
    1GB = 1024MB = 1,073,741,824B。
  6. 太字节（Terabyte，简写为TB）
    1TB = 1024GB = 1,099,511,627,776B。
  7. 拍字节（Petabyte，简写为PB）(往后都不常用)
    1PB = 1024TB = 1,125,899,906,842,624B。
  8. 艾字节（Exabyte，简写为EB）
    1EB = 1024PB = 1,152,921,504,606,846,976B。
  9. 泽字节（Zettabyte，简写为ZB）
    1ZB = 1024EB = 1,180,591,620,717,411,303,424B。
  10. 尧字节（Yottabyte，简写为YB）(普遍承认的最大的单位)
    1YB = 1024ZB = 1,208,925,819,614,629,174,706,176B。
  11. 鹤字节（Brontobyte，简写为BB）
    1BB = 1024YB = 1,241,249,140,039,285,880,159,884,800B
  12. 雷字节（Geopbyte，简写为GB）
    1GB = 1024BB = 1,270,319,627,481,870,963,251,417,600B
注意：上述的储存单位是以二进制计算的，也就是计算机常用的储存单位计算方式，但在现实生活中去购买储存设备一般都是按十进制计算的，所以现实中购买的储存的值比计算机中看到的要大(不是虚假内存)，实际容量(GB)乘1.073741824等于现实容量(GIB)

认识控件

完全启动系统后(欢迎界面结束后)看见的最大范围的控件叫做桌面(桌面是特殊的文件夹)。
桌面下方的长条叫做任务栏
任务栏上有很多按钮，其中微软商标图案的按钮点击后会打开开始菜单
开始菜单提供了一些基础功能，包括下载的程序
任务栏的右侧有小图标，一般称为小图标，用作展示程序员允许展示的后台程序(不允许且任务栏没有显示的在进行的软件多半是病毒(系统软件除外))
小图标右侧有"设备和打印机"按钮，用于管理连接的Windows的设备
日历可以查看时间规划任务，有一个像wechat发出的信息式的图标的按钮叫操作中心，或者通知中心，里面会展示软件的通知，包括系统检测的设备和杀毒情况
桌面左上角有几个图标，其中带箭头的表示快捷方式(在文件一节介绍)，当然，像自带的"此电脑(有的系统叫我的电脑或者计算机)"，"回收站"，"网络"等也是快捷方式，但不带箭头
所有文件夹里的东西都可以叫做文件(包括快捷方式)，文件点击一次叫做选择，双击打开
你把文件拖入回收站可以删除文件。回收站最大回收量为4GB，没有时间限制。选中文件后按delete也能删除文件，按住shift+delete可以永久删除文件。
双击文件之后打开的界面叫做窗体，窗体的左上侧一般显示程序的名字，有上册一般有三个按钮，发布是横线，正方形，x形，分别对应最小化，最大化和关闭窗口。

其他

在A软件运行一个文件可以叫做在A的环境下运行文件，比如说在命令行环境中运行代码，就是指一串代码在命令行中运行。
沙河是指没有某某程序提供一个隔离的环境，这个环境称为沙盒。

开关机

主机上有一个半圆加一个竖杠的标识符，他是二进制中0和1的组合图形，0代表关闭，1代表启动。
所以不要再说«我的电脑上只有开机键没有关机键»了
