选择你的电脑系统

首先，我们了解一下世界主流的电脑系统

  • Windows(微软)
    Windows 是全球使用最广泛的桌面操作系统。它由微软公司开发，常用于个人电脑、企业工作站和笔记本电脑。Windows 系统易于使用，支持大量的软件和硬件，兼容性高，也就是说，你对这台电脑的配置不满，你可以通过更换几乎任意厂家的芯片来升级此电脑。
    因为其使用最广泛，所以拥有最大的软件库，几乎所有软件都支持Windows，假如系统出现问题，可以很轻松地在网络上找到解决方案，微软官网售卖的电脑有超高的颜值，画在屏幕背面的中央的商标显得电脑非常气派。
    如果你希望获得好用，但这个软件比较小众(很多大众软件也不支持其他系统)，Windows是最好的选择
  • macOS (苹果)
    macOS 是苹果公司开发的操作系统，专门用于其Mac系列电脑。macOS 以其简洁、稳定而著称。它也提供了优异的图形和多媒体处理能力，常用于创意行业如设计、视频编辑等。
    但macOS系统封闭，对硬件兼容性不高，只能通过更换特定厂家的芯片升级配置，软件市场小，尤其是游戏，能支持的游戏(2024统计)可以说两只手就可以数过来
    和Windows一样，苹果官方的电脑颜值高，但macOS的动画优化优于Windows(视觉体验更好)，不过系统故障很难找到解决办法，目前(2024)据我了解，主要只有苹果自己售卖macOS
    但由于macOS软件市场远小于Windows，所以可以使用macOS系统来避免很多风险软件的安装(对于见过Windows的用户会很不习惯，比如其系统关闭浏览器要点击左侧红色点然后再任务栏右键图标，点击退出而不是点击右侧叉号，注意，macOS系统的鼠标滚轮是反的)
    对了，买这种系统之前记得先买一个转换器和一个扩展坞，因为不进行复杂的办公，所以一般不需要买鼠标。
  • Linux
    Linux 是一个开源的操作系统，广泛应用于服务器、嵌入式设备以及开发者和技术爱好者的个人电脑上。Linux 有许多不同的发行版（如 Ubuntu、Debian、Fedora、CentOS 等），并因其高定制性和强大的安全性在开发和服务器领域特别受欢迎。
    对于专业人士非常适合，但软件市场小(支持的软件比macOS多点，但仍然远远小于Windows)，系统没有官方售卖的电脑，最好在Windows系统上下载此系统，然后安装
  • BSD 系统（如 FreeBSD、OpenBSD、NetBSD）
    BSD 是一种类 Unix 操作系统，基于 Berkeley Software Distribution（BSD）。它具有高度的稳定性和安全性，广泛应用于服务器、路由器、嵌入式系统等领域。

以上是常见的四类电脑系统，如果你不知道改选什么，就直接选择Windows
如果你是仅看电视，使用基本的办公，且对视觉要求高，就选择macOS(屏幕好)。如果你专业开发或其他专业人士，选择Linux(推荐)或者BSD
如果都不是，选择Windows，否则你将放弃很多便利

当然，也可以组合使用，推荐使用Windows台式机(处理复制问题，当然，看电视，画图，剪辑等简单问题也可以)，加macOS笔记本(看电视，画图)。如果愿意，可以再准备一台电脑(笔记本台式都可以)使用Linux

鼠标

鼠标硬件篇

我们重点了解三个核心元件

首先是微动，微动是鼠标左右键下的元件，用来检测用户是否点击，假如你的鼠标按键失灵，可以选择更换微动，不一定要换整个鼠标

接着是滚轮，滚轮主要分两种，一种是"光栅式"，结构复制，价格较贵，但耐用，稳定，仅微软，罗技等少数企业生产。
另一种是"机械编码式"，其结构简单，便宜，但不耐用(一般是你的微动先坏，放心)。

传感器：传感器主要是用来识别所在位置，然后转换成数据汇报给计算机转换成光标移动的
对传感器的影响主要是鼠标垫，一般是编制布垫。当然也有好的传感器，比如微软蓝影技术(新一代鼠标追踪技术)，几乎可以用于任何地方。
鼠标传感器有两个参数。一个叫做采样率(CPI)。有人将其称为DPI(图像每英寸长度内的像素点数。)，但其实应该叫做CPI(英寸采样率)。通俗的讲，CPI越高，鼠标的灵敏度就越高。在现实中鼠标在鼠标垫上移动单位距离，CPI越大的鼠标在屏幕上移动的距离越大。CPI不是不能调，在系统的控制面板>硬件>鼠标选项 可以修改。 第二个参数叫做回报率，意思是指鼠标向计算机报告位置的频率。一般情况下频率越高，相对于的延迟越小。但过高的回报率与较高的回报率基本没有感受上的区别。

有些鼠标就有鼠标侧键。这些鼠标侧键跟键盘差不多,不同的鼠标侧键有不同的功能。但在游戏时使用这些鼠标侧键很容易被判定为作弊。

鼠标软件知识

鼠标左键在电脑中一般可以作为选择，比如你使用鼠标左键点击某一文件的时候，可以选中该文件。
鼠标右键可以理解为属性，在很多时候对着某东西使用鼠标右键会出现右键菜单。这些菜单里面一般提供了很多关于该文件属性的选项。
鼠标中键及鼠标滚轮，一般用作对过长的文件进行滚动。

鼠标左键选中文本之后拖动文本到某一可输入文本的地方，系统会复制该文本。(对于一些不允许复制的网站有效)。用鼠标将网站上的图片拖至文件夹或桌面(桌面也是一种文件夹。)，系统会下载该图片。在浏览器中，拖动网址进入输入框会直接填入该网址，拖入上方标签也会替换标签页或在标签页旁边插入新的标签页。

在浏览器上对着链接使用鼠标中键，可以使链接在后台新建标签页中打开。
在有标签页的软件上对着标签页使用鼠标中间可以快速关闭标签页。
使用Alt+Tab多任务视图时，可以使用鼠标中键对着任务视图单击关闭窗口。
在允许滚动的位置单击鼠标中键可以实现自动滚动，滚动的方向由你鼠标的移动方向决定。

鼠标组合键

该内容需要用到一点点键盘知识，你只需要知道这些按键在哪里就行。
左键

对文件使用双击，可以打开该文件。对文字使用双击，可以选中该词语。对文字使用三击，可以选中整行的文字。

在大多数情况下，你可以按住鼠标左键不放，然后拖动鼠标，就可以选中一片区域的文字或文件。也可以先对着起点单击鼠标左键。然后按住shift不放，最后在末尾单击鼠标左键，也可以达到选择一片区域的效果。

按住ctrl再使用鼠标单击链接，可以使链接在后台新建标签页中打开。

按住Alt在使用鼠标可以选中网站链接上的文字。

中键

按住ctrl使用鼠标滚轮滚动，可以在支持放大缩小的地方放大和缩小页面。

右键

按住shift加鼠标右键可以显示全部菜单。(contextmenumanager可以帮助你过来右键菜单。)

鼠标手势

鼠标手势是指使用鼠标按照特定的键位在滑动特定的轨迹实现特定的操作。

系统本不支持鼠标手势。但有些软件支持，比如说很多浏览器，部分输入法。也有专业的鼠标手势软件(mouselnc等)。
Quicker是一个强大的鼠标工具软件。提供了很多便捷功能。
