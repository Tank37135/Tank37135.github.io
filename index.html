内联HTML元素

HTML定义了一个可用内联标签的长列表，完整列表可以在Mozilla开发者网络上或者菜鸟教程上找到。

  • 加粗文本，使用<strong>。
  • 斜体文本，使用<em>。
  • 缩写，如HTML，应使用<abbr>，并可选择添加title属性以提供完整短语。
  • 引用，如— Mark Otto，应使用<cite>。
  • 删除的文本应使用<del>，插入的文本应使用<ins>。
  • 上标文本使用<sup>，下标文本使用<sub>。

这些元素大多数由浏览器以少量修改的样式显示。

这里，我们粗略地讲解一个表格一个表格的属性：

  • <table> 元素的属性
    • border：设置表格的边框宽度（已过时，建议使用 CSS）。

      例如：<table border="1"> 用于创建一个带有 1 像素宽边框的表格。

    • cellspacing：设置单元格之间的间距（已过时，建议使用 CSS `border-spacing`）。

      例如：<table cellspacing="5"> 用于设置单元格之间有 5 像素的间距。

    • cellpadding：设置单元格内容与边框之间的内边距（已过时，建议使用 CSS `padding`）。

      例如：<table cellpadding="10"> 用于设置单元格内内容与边框之间的间距为 10 像素。

    • width：设置表格的宽度，可以使用具体的像素值或百分比。

      例如：<table width="100%"> 用于设置表格宽度占据父容器的 100%。

    • height：设置表格的高度。

      例如：<table height="400px"> 用于设置表格高度为 400 像素。

    • align：设置表格在页面中的水平对齐方式（已过时，建议使用 CSS `text-align`）。

      例如：<table align="center"> 用于将表格居中显示。

    • summary：为表格提供简短的说明，主要用于无障碍设计。

      例如：<table summary="This table shows sales data for Q1"> 用于为表格提供简要描述，方便屏幕阅读器用户。

  • <tr> 元素的属性
    • align：设置表格行内容的水平对齐方式（已过时，建议使用 CSS）。

      例如：<tr align="center"> 用于设置行中的内容居中对齐。

    • valign：设置表格行内容的垂直对齐方式（已过时，建议使用 CSS）。

      例如：<tr valign="top"> 用于将行中的内容垂直对齐到顶部。

  • <td> 元素的属性
    • colspan：设置单元格跨越的列数。

      例如：<td colspan="2">合并两列</td> 用于合并两列。

    • rowspan：设置单元格跨越的行数。

      例如：<td rowspan="2">合并两行</td> 用于合并两行。

    • align：设置单元格内容的水平对齐方式。

      例如：<td align="right"> 用于将单元格内容右对齐。

    • valign：设置单元格内容的垂直对齐方式。

      例如：<td valign="middle"> 用于将单元格内容垂直居中。

    • width：设置单元格的宽度。

      例如：<td width="100px"> 用于设置单元格的宽度为 100 像素。

    • height：设置单元格的高度。

      例如：<td height="50px"> 用于设置单元格的高度为 50 像素。

  • <th> 元素的属性
    • colspan：设置标题单元格跨越的列数。

      例如：<th colspan="2">标题</th> 用于合并两列的标题单元格。

    • rowspan：设置标题单元格跨越的行数。

      例如：<th rowspan="2">标题</th> 用于合并两行的标题单元格。

    • align：设置标题单元格内容的水平对齐方式。

      例如：<th align="center">标题</th> 用于将标题单元格的内容居中对齐。

    • valign：设置标题单元格内容的垂直对齐方式。

      例如：<th valign="bottom">标题</th> 用于将标题单元格的内容垂直对齐到底部。
